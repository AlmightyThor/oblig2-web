Git og GitHub

Git er en programvare hvor du kan lage et git respository hvor du kan legge inn alle kildefiler og versjoner man lager. Her ser man ulike "commits" av koden, og man kan se ulike bilder fra når koden var laget.

GitHub er et webhotell hvor man kan publisere Git repository sitt og samarbeide med andre personer.
Et eksempel på fordelen med dette er at hvis man kommer over en error mellom "committene", kan man se forskjellen mellom koden som den er nå og siden den sist fungerte, for å hjelpe deg med å finne feilen.
